# Szybkie przykłady NoSQL w Pythonie

W paczce są dwa bardzo krótkie przykłady:

1. `key_value_example.py`
   - symulacja bazy key-value na słowniku Pythona
   - pokazuje zapis, odczyt, aktualizację i usunięcie danych

2. `wide_column_example.py`
   - uproszczona symulacja bazy kolumnowej / wide-column
   - pokazuje dane partycjonowane po użytkowniku i sortowane po dacie
   - przykład z historią zamówień użytkownika

Uruchomienie:
```bash
python key_value_example.py
python wide_column_example.py
```
